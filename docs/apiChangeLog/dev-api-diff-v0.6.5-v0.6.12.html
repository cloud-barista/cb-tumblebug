Changelog

Versions ⇑

Changes from latest to latest.

What's New ⇑

  1. GET/ns/{nsId}/mcis/{mcisId}/subgroup List SubGroup IDs in a specified MCIS
  2. GET/ns/{nsId}/mcis/{mcisId}/subgroup/{subgroupId} List VMs with a SubGroup label in a specified MCIS
  3. POST/ns/{nsId}/mcis/{mcisId}/subgroup/{subgroupId} ScaleOut subGroup in specified MCIS
  4. GET/ns/{nsId}/mcis/{mcisId}/vm/{vmId}/dataDisk Get available dataDisks for a VM
  5. PUT/ns/{nsId}/mcis/{mcisId}/vm/{vmId}/dataDisk Attach/Detach available dataDisk
  6. POST/ns/{nsId}/mcis/{mcisId}/vm/{vmId}/snapshot Snapshot VM and create a Custom Image Object using the Snapshot
  7. GET/ns/{nsId}/resources/customImage List all customImages or customImages' ID
  8. POST/ns/{nsId}/resources/customImage Register existing Custom Image in a CSP
  9. DELETE/ns/{nsId}/resources/customImage Delete all customImages
  10. GET/ns/{nsId}/resources/customImage/{customImageId} Get customImage
  11. DELETE/ns/{nsId}/resources/customImage/{customImageId} Delete customImage

What's Deprecated ⇑

  1. PUT /ns/{nsId}/mcis/{mcisId}/vm/{vmId}/{command} Attach/Detach data disk to/from VM
  2. GET /ns/{nsId}/mcis/{mcisId}/vmgroup List VMGroup IDs in a specified MCIS
  3. POST /ns/{nsId}/mcis/{mcisId}/vmgroup Create multiple VMs by VM group in specified MCIS
  4. GET /ns/{nsId}/mcis/{mcisId}/vmgroup/{vmgroupId} List VMs with a VMGroup label in a specified MCIS
  5. POST /ns/{nsId}/mcis/{mcisId}/vmgroup/{vmgroupId} ScaleOut VM group in specified MCIS
