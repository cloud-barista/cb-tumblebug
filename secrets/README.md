# Secrets Directory

This directory stores sensitive files generated during OpenBao initialization.

## Files

| File                | Description                          | Generated by           |
| ------------------- | ------------------------------------ | ---------------------- |
| `openbao-init.json` | Unseal keys + root token             | `init/init-openbao.sh` |
| `.keep`             | Placeholder to keep directory in Git | —                      |

> **IMPORTANT**: `openbao-init.json` contains the unseal key and root token.
> It is excluded from Git via `.gitignore`. Keep this file safe.

For initialization instructions, see [init/README.md](../init/README.md).
